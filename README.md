# Salon Appointment System (Frontend Only)

## Features
- Services list + category filter
- Book appointment (service, staff/any staff, date, slot, customer form)
- Prevents double booking per staff (time overlap check)
- Auto-assign staff when "Any staff" is selected
- My Bookings search by Booking ID or Phone + cancel
- Admin dashboard + filter + cancel
- Uses localStorage (no backend)

## Run
1) Install dependencies
```bash
npm install
```

2) Start dev server
```bash
npm run dev
```
